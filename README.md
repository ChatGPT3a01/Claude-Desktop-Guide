# Claude Desktop 桌面版完全指南

從安裝到精通的實戰教學課程

## 課程簡介

本課程專為想要學習 Claude Desktop 桌面應用程式的使用者設計，採用「做中學」的實作導向教學方式，每個單元都包含豐富的實際操作範例。

## 線上觀看

**GitHub Pages**: https://chatgpt3a01.github.io/Claude-Desktop-Guide/簡報/

## 課程大綱

| Part | 主題 | 內容 |
|------|------|------|
| 1 | 安裝與基本設定 | 下載安裝、帳號登入、基本設定、全域快捷鍵 |
| 2 | 介面導覽與基本對話 | 主介面、快捷鍵、對話管理、歷史搜尋 |
| 3 | 專案功能實作 | 建立專案、設定指令、知識庫、對話管理 |
| 4 | 文件處理實作 | PDF 分析、文件摘要、表格處理、多檔比較 |
| 5 | 程式碼協助實作 | 程式生成、除錯、解釋、優化、語言轉換 |
| 6 | 寫作與翻譯實作 | 商業寫作、翻譯、風格轉換、文字潤飾 |
| 7 | MCP 伺服器設定 | MCP 介紹、設定檔、常用伺服器 |
| 8 | 檔案系統整合 | 讀取檔案、搜尋、整理、批次處理 |
| 9 | 進階應用實作 | 工作流程自動化、數據分析、專案管理 |
| 10 | 最佳實踐與技巧 | 提示技巧、效率提升、常見問題 |

## 課程特色

- 實作導向：每個單元都有實際操作範例
- 循序漸進：從基礎到進階系統性學習
- 即學即用：學完馬上能應用到工作中
- 中文教學：全繁體中文，台灣用語

## 系統需求

- **Windows**: Windows 10 以上
- **macOS**: macOS 12 以上
- **Node.js**: v18 以上（MCP 功能需要）
